# **App Name**: Vedro: Learning Through Interaction

## Core Features:

- Login/Signup with Scroll UI: Secure authentication system with email, password, and Google Auth, styled with an ancient scroll UI.
- AI Chatbot (Vedro AI): An AI-powered chatbot that provides doubt solving, explanations, summaries, and MCQs, integrated with Gemini API and Quill writing animation, and acts as a tool which could determine if additional resources may be needed.
- Concept Builder Game: A drag-and-drop game to arrange steps in the correct order, featuring a wooden tile theme and gold glow animation on correct placement.
- Sandbox Experiment: A free experiment zone where students can drag and drop items to simulate reactions, with AI explanations and animations.
- Running Quiz Game: An infinite runner game with three path choices, where correct answers trigger a gold flare and wrong answers trigger a red ink splash, including a scoring system and increasing difficulty.
- Profile Management: User profile page with name, email, game history, scores, badges, and theme toggle.
- Firebase Backend Integration: Cloud Functions for AI responses, game evaluation, user activity logging, score updates, and experiment result storage using Firestore.

## Style Guidelines:

- Primary color: Dark Brown (#5C3A21) for a rich, ancient feel.
- Primary color: Ochre (#C49E54) evokes an old paper vibe
- Primary color: Parchment Yellow (#E6D7B8) imitates aging paper.
- Secondary color: Deep Burnt Brown (#2A1B10) is a background element for contrast.
- Secondary color: Ancient Ink Red (#8B3A2B) for highlights to show important details.
- Highlight color: Soft Gold (#D4B46E) for gold features like winning or collecting, etc.
- Highlight color: Dusty Sand (#C8B58A) as an alternative gold look.
- Body and headline font: 'Literata' serif for a literary and vintage feel. 
- Scroll Unroll animation for page transitions, mimicking old scrolls opening.
- Torch Flicker Glow animation to provide an ambient golden flicker, creating a magical classroom atmosphere.
- Ink-Fade Text animation to make text appear blurry before sharpening, adding a unique visual effect.
- Burnt-Edge Pulse animation to make borders glow, creating a dynamic user interface.
- Dust Particle Drift animation for floating particles, enhancing the app's immersive environment.
- Quill Writing animation to simulate messages being typed from left to right, adding a personal touch.
- Wax Seal Press animation for button presses, creating a satisfying tactile feedback.
- Custom icons with an ancient parchment and torch-lit style, enhancing the app's theme.
